<h1># C---Boat-Renting-Calculator</h1>

<h2>Description</h2>
This code is programmed to be able to calculate and give the price regarding what boat you want to rent and how many life jackets you will need. The calculations are based off the price of each of the different boats, the amount of days, how many people, if you're of age, and if you want to purchase any life jackets.

- <b>Visual Studio</b>
- <b>C#</b>
